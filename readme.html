Příručka ovládání aplikace do PDB

Databáze se inicializuje spuštěním aplikace s parametrem -i nebo --init. Tato inicializace vytvoří tabulky a naplní je daty.

Překlad a spuštění

Nejprve je nutné nastavit v souboru build.properties cestu k Java JDK (1.7).
Dále je nutné stáhnout všechny potřebné knihovny z http://www.stud.fit.vutbr.cz/~xpracu00/libs.zip a archiv rozbalit do hlavního adresáře projektu.
Pak je možný překlad pomocí nástroje Ant: spustit příkaz ant bez parametrů.
Pro spuštění aplikace: ant run.
Pro spuštění aplikace s inicializací databáze: ant run_init.

Ovládání Prostorové části

Pro vložení nové enity kliknout do mapy levým tlačítkem myši tam, kde se má vytvořit, a pak vybrat typ entity a kliknout na tlačítko New.
Při vybraném bodu entity lze pravým tlačítkem myši přidat další bod. S body je možné manipulovat levým tlačítkem myši.
Při vybraném bodu entity, nebo při vybrané entitě je lze smazat prostředním tlačítkem myši.
Změny se ukládají tlačítkem Save All.

Ovládání Multimediální části

Výběrem řádku v tabulce se zobrazí obrázek.
Dvojitým poklikáním na jméno obrázku ho lze přejmenovat, a pak tuto změnu uložit tlačítkem Save.
Nápověda funkcí tlačítek se zobrazuje při najetí myší na tlačítka.
